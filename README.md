# BaseLaunch

A cohort-based developer platform for learning Web3 development on Base network.

## Features

- Browse available developer cohorts
- View detailed cohort information
- Enroll in cohorts (coming soon)

## Tech Stack

- Next.js 14 with App Router
- Tailwind CSS
- Supabase (PostgreSQL)
- Base network with OnchainKit
